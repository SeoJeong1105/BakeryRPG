## 2. 개발 환경
* Unity
* C#

## 3. 사용 기술
|기술|설명|
|:---------:|:-----------:|
|Singleton|Manager, Inventory 관리|
|Obejct Pooling|과일, 몬스터의 오브젝트 재사용|
|Scriptable Object|아이템, 메뉴를 Prefab화하여 메모리 절약|
